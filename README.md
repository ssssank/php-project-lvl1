## Description:
Collection of mini games for the console. To win you need to answer three times correctly.

## Installation:
```bash
git clone alegit@github.com:ssssank/php-project-lvl1.git

make install
```

# Use:
```bash
make brain-even
make brain-calc
make brain-gcd
make brain-progression
make brain-prime
```
